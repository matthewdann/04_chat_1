# 課題04 Firebase
## タイトル
- なんちゃってボット
## 説明
- 名前とメッセージを入力し送信すると、表示領域に名前、メッセージ、送信日時、削除ボタンが表示される。それと同時にアプリが返事をする。アプリの返事はあらかじめ用意されているものからランダムに選ばれる。
## 工夫した点
- 改行を入力すると送信するようにした。
- ただし、Shift + Enterの場合は送信せず、改行して入力・表示されるようにした。
- 常に最新のメッセージが見えるようにした。
## 解決したいこと
- ユーザーのメッセージが右にアプリのが左に出るようにしたかったが、アプリのも右に出てしまうことがある。
